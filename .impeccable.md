## Design Context

### Users
Homespire Home Loans loan officers. They check this dashboard for current mortgage rates before and between client calls — quick, high-frequency glances on both desktop and mobile. The tool needs to deliver confidence: an LO pulling this up mid-conversation with a borrower should feel like they have a sharp, authoritative data source, not a toy.

### Brand Personality
**Premium, confident, precise.** Homespire positions itself as a trustworthy lender ("Your journey home, easier"). The interface should feel like a well-made instrument — something a professional reaches for because it's fast, clear, and reliable. Not flashy, not playful. Quiet authority.

### Aesthetic Direction
- **Tone**: Clean and refined. Strip away glassmorphism, ambient orbs, and decorative glow in favor of tight typography, generous whitespace, and sharp data hierarchy. Think Bloomberg meets a luxury watch catalog — information-dense but beautifully composed.
- **Theme**: Dark mode default (monitoring context — LOs scanning rates throughout the day). Light mode available as toggle.
- **Brand colors**: Primary purple `#67489D`, secondary `#7C5DB2`, accent glow `#D6BEFF`. Green/red for rate movement indicators.
- **Anti-references**: No neon-on-dark "crypto dashboard" aesthetics. No glassmorphism cards. No decorative sparklines or ambient blob animations. No generic SaaS admin panel look.
- **References**: The density of a financial terminal, the restraint of a Dieter Rams product, the typography of a premium print annual report.

### Design Principles
1. **Data first, decoration never.** Every visual element must serve the data. If it doesn't help an LO read a rate faster or understand a trend clearer, remove it.
2. **Quiet confidence.** Premium feels restrained, not loud. Use spacing, type weight, and subtle color to create hierarchy — not glow, blur, or animation excess.
3. **Scannable in 2 seconds.** An LO glancing at their phone between calls should get the 30yr fixed rate and its direction in under 2 seconds. Optimize for that moment.
4. **Professional credibility.** This tool gets shown to borrowers. It should look like it belongs at a serious financial institution, not a startup hackathon.
5. **Motion with purpose.** Animate only to communicate state changes (data refresh, rate movement). Entrance animations should be fast and subtle, not theatrical.
